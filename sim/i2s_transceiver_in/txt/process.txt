Assumptions:
using 24 bit signed data (MSB is sign)
data source correctly used WS
clock is gonna come from my testbench

## HIGH-LEVEL

1. take in 24 bits

2. load up a [23:0] register, latching on WS, counting bits along the way

3. switch registers on WS change for stability

3. output a register when full, sets a flag high (headed to a FIFO module for storage)


## DETAILED

0. Reset logic: clear registers, set data flag - 0;

1. Latch WS, SD on posedge SCK

2. If WS changed → reset counter, clear register

3. Shift SD bit into register[23−counter]

4. Increment counter

5. When counter = 24 → assert output_valid, latch register to FIFO
